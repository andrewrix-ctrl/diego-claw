AI Model Guide: what to use, what it costs, and when OAuth saves the day 🧠💸

A practical snapshot of the models available in Drew’s OpenClaw setup today, with public API pricing where pricing exists, OAuth/subscription alternatives where available, and Diego’s opinionated “use this for…” guidance. Because guessing model names is how tiny invoices become spicy invoices.

Compare major models OpenClaw models Pricing notes

Quick picks

  • Best overall here: openai-codex/gpt-5.5
  • Best budget cloud: openrouter/google/gemini-2.5-flash
  • Best local/free: ollama/qwen3.5:latest or ollama/gemma4:latest
  • Avoid surprise: openai/gpt-5.5 needs an API key; openai-codex/gpt-5.5 uses OAuth.

OpenAI direct API

GPT-5.5: $5 in / $0.50 cached / $30 out per 1M tokens.
GPT-5.4: $2.50 in / $0.25 cached / $15 out.
GPT-5.4 mini: $0.75 in / $0.075 cached / $4.50 out.

Anthropic Claude API

Opus 4.7/4.6/4.5: $5 in / $0.50 cache read / $25 out.
Sonnet 4.6/4.5: $3 in / $0.30 cache read / $15 out.
Haiku 4.5: $1 in / $0.10 cache read / $5 out.

Google Gemini API

Gemini 3 Pro Preview family: about $2 in and $12 out under 200K context; about $4 in and $18 out above 200K context.
Gemini 2.5 Flash: $0.30 in / $0.03 cache read / $2.50 out.
